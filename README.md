⬡ TecnoVida — Tecnologia & Comunidade


Plataforma web educacional que explora como inovação, sustentabilidade e cooperação podem transformar comunidades e construir um futuro mais justo e humano.




📋 Sobre o Projeto

TecnoVida é um site fictício de caráter educacional criado para demonstrar como a tecnologia pode ser aplicada ao bem-estar coletivo. A plataforma apresenta conteúdos sobre educação digital, sustentabilidade ambiental, inovação social e qualidade de vida, de forma visualmente rica e interativa.

🛠️ Tecnologias Utilizadas


HTML5 — estrutura semântica com roles ARIA para acessibilidade
CSS3 — animações, variáveis CSS, grid/flexbox, design responsivo
JavaScript Vanilla — sem dependências externas



✨ Funcionalidades


Navbar responsiva com efeito de scroll, menu hambúrguer mobile e destaque de seção ativa
Scroll suave com âncoras e offset automático para o header fixo
Reveal on scroll — elementos animam ao entrar na viewport via IntersectionObserver
Contadores animados — números sobem progressivamente ao carregar a página
Cards de curiosidades com efeito flip ao clicar
Quiz interativo com barra de progresso, feedback por questão e pontuação final
Formulário de contato com validação de campos e mensagem de sucesso
Botão "Voltar ao topo" que aparece ao rolar a página
Acessibilidade — uso de aria-label, aria-expanded, role, aria-live em toda a marcação

🚀 Como Executar

Por ser um projeto estático (apenas HTML, CSS e JS), basta abrir o arquivo index.html diretamente no navegador:

# Clone ou extraia o projeto
cd TecnoVida-main

# Abra no navegador (Linux/Mac)
open index.html

# Ou use uma extensão como Live Server no VS Code para servir localmente


📱 Responsividade

O layout é totalmente responsivo, adaptando-se a:


Desktop — grid multi-colunas, hero com visual lateral
Tablet — colunas reduzidas, espaçamentos ajustados
Mobile — coluna única, menu hambúrguer, navegação touch-friendly
